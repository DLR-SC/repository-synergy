A script that simulates a "hacking program", as often seen in movies.

[Original code by janbrennen](https://github.com/janbrennen/rice/blob/master/hack.exe.c)

To install:

`pip install hack-exe`

To run:

`hack.exe <URL>`

e.g.:

`hack.exe google.com`
